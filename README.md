# yoloDetection

## Steps
### 1. Clone to local
```sh
git clone https://github.com/newjoy2018/yoloDetection.git
cd yoloDetection
```
### 2. Put your dataset here
Put your dataset into directory **yoloDetection**, and rename it as **KITMoMa**

### 3. Choose model and Run
```
chmod +x run.sh
./run.sh
```
